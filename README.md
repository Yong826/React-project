
# 1. 프로젝트 Description

## - 애니매이션 위주와 반응형 목적으로 연습용으로 제작한 웹 사이트 입니다.

# 2. 프로젝트 정보

## - 프로젝트는 Window 10 기준으로 제작되었고 Tool 은 VS code를 사용하였습니다.
## - 반응형 웹 그리고 그리드 목적으로 BootStrap을 사용했습니다.
## - 사용한 기술스택은 React 입니다.
